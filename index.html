🗄️ Banco de Dados (Supabase)

Conecte ao Supabase para salvar na nuvem e compartilhar com sua equipe. Quem tiver a URL e a chave consegue ver e editar os dados.

Setup rápido:
1. Acesse supabase.com → crie projeto gratuito
2. Settings → API: copie Project URL e anon key
3. No SQL Editor execute:

CREATE TABLE IF NOT EXISTS clientes (id TEXT PRIMARY KEY, dados JSONB);
CREATE TABLE IF NOT EXISTS historico (id BIGSERIAL PRIMARY KEY, entrada JSONB);
ALTER TABLE clientes ENABLE ROW LEVEL SECURITY;
ALTER TABLE historico ENABLE ROW LEVEL SECURITY;
CREATE POLICY "all" ON clientes FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "all" ON historico FOR ALL USING (true) WITH CHECK (true);
ALTER PUBLICATION supabase_realtime ADD TABLE clientes;
ALTER PUBLICATION supabase_realtime ADD TABLE historico;
